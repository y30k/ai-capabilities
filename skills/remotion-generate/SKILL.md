# Remotion Generate

Use this skill for Remotion composition creation, modification, preview, and render tasks.

Read `references/workflow.md` before editing compositions or running render commands.

## Documentation Output

When writing plans, reports, PRDs, briefs, findings, story tracking, scratch notes, or other generated documentation, write them under the repository-root `docs/` directory, preferably `docs/remotion-generate/...` or the specific `docs/` path named in the workflow. Do not use `.agents/`, `.pi/`, `.codex/`, `.claude/`, or other agent-specific directories for generated documentation.
